# Système de Réservation de Salles - ÉTS

Application web permettant aux étudiants de l'ÉTS de réserver des salles d'étude.

## Technologies utilisées
- Node.js / Express
- MySQL
- bcrypt, express-session

## Fonctionnalités
- Inscription et connexion sécurisée
- Consultation des salles disponibles
- Réservation avec détection de conflits
- Annulation de réservation

## Installation
1. Cloner le repo
2. `npm install`
3. Créer un fichier `.env` avec vos credentials MySQL
4. Lancer avec `node server.js`
